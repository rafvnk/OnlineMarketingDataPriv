Browser Fingerprint

March 22, 2021 by Rafael Von-Klaveren

A browser fingerprint is a unique identifier that is generated to be used to uniquely identify you amongst all internet users. Web pages are composed of HTML, CSS, and Javascript. One of the most powerful tools that is currently being used now is Javascript. If a website has Javascript code running in the background it has the ability to know which browser you’re currently using, OS, device type, and even the model of the device alongside with the timezone you’re currently in, plug ins installed within the browser and even allows it to monitor if your cursor is moving. All of these bits of information is added into the full unique identifier within the browser fingerprint.

How dangerous is this? Well, Scottie from ScottiesTech.Info invested a day working on a browser fingerprinting code from scratch. In one day on working with that code he was able to get it up and running with it uniquely identifying one in eight and a half billion people, which is currently more people on planet earth. This shows the power of browser fingerprints. If in one day, one guy was able to create a powerful tool like that, imagine what these big companies are able to do with a full team and a full month of working on a browser fingerprint.

Online Privacy

March 22, 2021 by Rafael Von-Klaveren

We know nothing is free. Usually when we use “free” online services we hit accept on the terms of service without even thinking about it. When we use these free online services we’re usually tagged as the product. Not us in the physical form, but the virtual data we contribute to the internet. Each time we access a site, like a picture, post a tweet you’re essentially leaving a breadcrumb of info about your online presence available.

Let’s say if you’re someone who keeps a very low profile online. You rarely use social media, engage in online communities, or keep your online usage very low. If someone has access to only your name they’ll still be able to grab data that is made public online. Birthday, home address, cars you own, emails you’ve used in the past, literally everything. One breadcrumb is all people need to essentially obtain this info that can lead to your physical presence.
